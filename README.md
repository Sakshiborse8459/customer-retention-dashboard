# 📊 Customer Engagement & Product Utilization Analytics for Retention Strategy

## Overview
This project analyzes customer engagement, product utilization, and churn behavior in the banking sector using data analytics and business intelligence techniques. An interactive Streamlit dashboard was developed to help identify customer retention trends, high-risk customers, and opportunities for improving customer loyalty.

## Objectives
- Analyze customer engagement and churn patterns.
- Evaluate product utilization and customer behavior.
- Identify high-value customers at risk of churn.
- Build an interactive dashboard for business decision-making.
- Recommend data-driven retention strategies.

## Dataset
The project uses a banking customer dataset containing:
- Customer Demographics
- Credit Score
- Geography
- Age
- Balance
- Number of Products
- Customer Activity
- Estimated Salary
- Churn Status

Additional engineered features include:
- Engagement Segment
- Relationship Strength Score
- Sticky Customer
- High Value Customer
- Product Category
- Risk Score

## Key Insights
- Inactive customers have a significantly higher churn rate.
- Customers with multiple products show better retention.
- High-value but disengaged customers require targeted retention strategies.
- Customer engagement is a strong indicator of long-term loyalty.

## Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/customer-retention-dashboard.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
